# Number Battle Game

A simple JavaScript game where the user competes against the computer by generating random numbers.

## Live Demo

https://alexandrabalinskaiya.github.io/lr8/

## Description

Number Battle Game is a small frontend practice project built with HTML, CSS and JavaScript.

The user enters a name, then plays against the computer for three rounds. In each round, both sides receive a random number from 1 to 10. The side with the higher number gets a point. After three rounds, the game shows the final result.

## Features

* random number generation
* user name input
* player vs computer logic
* round counter
* score comparison
* final result message
* game reset after the result

## Technologies

* HTML5
* CSS3
* JavaScript

## How to Use

1. Open the live demo.
2. Enter your name.
3. Click the Generate button.
4. Play three rounds against the computer.
5. The game will show who won.
